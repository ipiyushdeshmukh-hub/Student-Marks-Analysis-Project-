# Student Performance Analysis

This project analyzes student marks using Python (Pandas + Matplotlib).

## Features
- Read CSV data of student performance
- Calculate average, max, min scores
- Generate bar charts, box plots, heatmaps
- Attendance vs performance analysis
- Risk level prediction based on marks

## How to Run
1. Install required libraries:
   pip install -r requirements.txt

2. Open Jupyter Notebook:
   analysis.ipynb

## Dataset Used
StudentsPerformance.csv
